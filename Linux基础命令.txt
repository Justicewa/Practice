目录命令:ls -l  -a

pwd	查看当前所在路径 例/home/dev(绝对路径:基于根目录的路径表达) 最前面的"/"叫根目录,其余的"/"叫分隔符
	Linux下的目录结构:两个分区:交换分区(交换内存,内存不够用了就把内存上的不活跃数据放到交换分区)
				  文件系统分区(可存在多个)(给某个目录分配空间,因为先有目录再有空间)
			"."当前路径".."本路径的上一层路径(相对路径:相对于当前路径的表达方式")每个目录下都会有它俩
mkdir +　目录名　创建目录
mkdir -p abc/bcd/cde 多层级创建目录

rmdir workspace  清理空目录
rmdir -p abc/bcd/cde 递归删除空目录
rm 删除普通文件 rm -rf ./*(删完)
    -r 递归删除目录下所有文件,包括自身
    -f 忽略提示信息,直接进行操作
    /etc/* 
cp test/ 到目标目录 拷贝一个文件到指定位置
    -r 递归拷贝目录下所有文件包括自身
    -f 忽略提示信息,直接进行操作提示信息
mv + 要移动的文件 + 目标目录下 将一个文件或目录移动到指定目录
    -i 增加
cd 修改当前所在路径为指定路径
   "~"返回家目录(随用户改变)
   "-"返回上一次所在路径

文件命令:
touch 刷新文件时间属性;文件不存在,则创建空的新文件
	-d 使用指定的时间刷新文件的时间属性
	touch abc.txt

cat 将文件内容打印到终端显示中
    -n 加入行号
	cat filename

tac 按行逆序打印文件内容

more 分页显示内容 回车是下一行 空格是下一页 q是退出

less 分页显示 f/b翻页 jk按行滚动 
	/string 向下匹配
	?string 向上匹配
	按n匹配下一个

head 默认显示文件的前10行内容
	-n n 文件名 显示文件的前n行内容
	打印11行内容: head -n 11 文件名 | tail -n 1 

tail 默认显示末尾10行内容
	-n n
	-f 一直动态刷新文件末尾内容 
	ctrl c  退出

>  清空重定向:清空原有的问价,写入新文件
